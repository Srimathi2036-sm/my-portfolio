# 🚀 Portfolio Website

This is my personal portfolio website built using modern web technologies to showcase my projects, skills, and achievements.

---

## 🛠️ Setup Instructions

Follow the steps below to run the project locally:

### 1️⃣ Install Dependencies

Install all required packages in both frontend and backend folders:

```bash
npm install
```

> Run this command inside both the **frontend** and **backend (server)** directories.

---

### 2️⃣ Run Backend Server

Navigate to the backend folder and start the server:

```bash
cd server
node server.js
```

✔ This will:

* Start the backend server
* Connect to MongoDB database

---

### 3️⃣ Run Frontend

Navigate to the frontend folder and run:

```bash
npm run dev
```

✔ This will:

* Start the frontend development server
* Launch the portfolio website in your browser

---

## 💻 Tech Stack

* Frontend: React.js, Tailwind CSS
* Backend: Node.js, Express.js
* Database: MongoDB

---

## 🌟 Features
